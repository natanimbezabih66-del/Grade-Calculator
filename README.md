# Grade-Calculator

A simple Java application that calculates a student's letter grade based on their score.

## Features

- Accepts a score from 0 to 100
- Calculates the corresponding letter grade
- Uses conditional statements (if-else)

## Technologies Used

- Java
- Scanner Class

## How to Run

1. Compile the program

```bash
javac GradeCalculator.java
```

2. Run the program

```bash
java GradeCalculator
```

## Example

Input:

```text
Enter your mark (0-100): 85
```

Output:

```text
Grade: B
```
